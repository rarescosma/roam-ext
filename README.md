Roam Export Extension
===

Dumb chrome extension that exports the current [Roam](https://roamresearch.com/) 
page by clicking around in the UI.

Default keyboard shortcut: <kbd>Alt</kbd> + <kbd>Shift</kbd> + <kbd>E</kbd>
